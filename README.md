📌 Project Overview

This project presents an Excel-based sales dashboard designed to analyse sales performance by product type and model.
The dashboard provides clear and interactive visualisations to support data-driven decision-making.

🎯 Key Objectives

-Analyse sales performance across different product types and models

-Identify top-selling and most profitable products

-Track overall sales trends

-Provide an easy-to-read business summary using KPIs

📈 Key Metrics

-Total Quantity Sold

-Turnover (Revenue)

-Profit

-Discount Rate

-Unit Price

-Purchase Price

🛠 Tools Used

-Microsoft Excel

-Pivot Tables

-Charts & Data Visualisation

-Dashboard Design

-Download the Excel file from this repository
